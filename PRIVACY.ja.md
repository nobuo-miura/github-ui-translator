# プライバシーポリシー — GitHub UI Translator

[English](PRIVACY.md) | [日本語](PRIVACY.ja.md)

最終更新日: 2026年7月11日

## 要約

**GitHub UI Translator は、個人情報や機密性の高いユーザーデータを収集・送信・販売しません。翻訳のON/OFF状態と言語設定のみを端末内に保存します。**

## 詳細

### データ収集は行いません

本拡張機能は、個人情報・閲覧履歴・ページの内容・利用状況などを一切収集しません。テレメトリ（利用統計の送信）もありません。

### 外部通信は行いません

翻訳はすべて、拡張機能に同梱された辞書ファイルを使ってブラウザ内で完結します。外部サーバー・翻訳API・クラウドサービスへのネットワーク通信は一切行いません。閲覧中のページの内容がブラウザの外に出ることはありません。

### ページ内容の処理はブラウザ内で完結します

本拡張機能は、翻訳のためにGitHubページ上のUI文言（ラベル・ボタン・メニュー項目など）をブラウザ内で一時的に参照します。ページ内容を記録・保存・外部送信することはありません。

### 設定はブラウザ内にのみ保存されます

本拡張機能が保存するのは「翻訳のON/OFF状態」と「選択言語」の2つの設定のみで、ブラウザの拡張機能用ストレージ（`chrome.storage.local` / `browser.storage.local`）を使用します。これらの設定は端末内にのみ保存され、どこにも送信されません。

### 権限について

- **storage**: 上記のON/OFF状態と言語選択を保存するためだけに使用します。
- **`https://github.com/*` へのホスト権限**: GitHubのページ上で翻訳スクリプトを動作させるために必要です。同梱辞書によるUI文言の置き換えと、設定変更時のGitHubタブの再読み込みにのみ使用します。

### 第三者への提供はありません

データが端末の外に出ることがないため、第三者への共有・販売・処理の委託は一切ありません。

## ポリシーの変更について

本ポリシーに変更がある場合（例: 将来のバージョンでオプションのオンライン機能を追加する場合など）は、適用前に本ページおよびリリースノートで告知します。
